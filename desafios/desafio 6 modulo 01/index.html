Links

  • Cria um link e iclui atributos em comum
    1. href - E-mail: mailto e link externo:http
    2. name - Nome da âncora
    3. target - Janela ou local que deve ser aberto: _blank, _slef, _top, _parent
    4. rel – Define os tipos de link que avançam
    5. rev – Define os tipos de link que revertem a ação
    6. acesskey – Atribui uma tecla de atalho para este elemento
    7. shape – Para uso com formas de objeto
    8. coords – Para uso com formas de objeto
    9. tabindex – Determina a ordem das guias
    10. onclick – É um evento JavaScript
    11. onmouseover – É um evento JavaScript
    12. onmouseout – É um evento JavaScript
